|----|------|

| 样式 | TailwindCSS |
| API 层 | tRPC |
| 数据库 | SQLite |
| ORM | Drizzle ORM |
| 部署 | Manus Hosted（绑定 manus.space） |

---

## 核心功能

- 个人目标创建与管理
- 每日打卡记录与连续打卡统计
- 司马黑吉祥物监督系统（毒舌激励语录）
- 孟菲斯设计风格 UI（明亮几何色块）
- 打卡数据可视化（连续天数、完成率）
