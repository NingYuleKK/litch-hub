## 技术栈

本项目托管于 Manus 平台，采用 Manus 内置的 web-static 脚手架构建。

| 层 | 技术 |
|----|------|
| 前端框架 | Vite + React + TypeScript |
| 样式 | TailwindCSS |
| 部署 | Manus Hosted（绑定 litchnotes.com） |
| 评论系统 | Giscus（GitHub Discussions 驱动） |
| 订阅通讯 | Buttondown |

---

## 核心功能

- 金瓶梅章回笔记的展示与归档
- 深夜书房暗色主题，沉浸式阅读体验
- 章节批注与研究笔记系统
- Newsletter 订阅（Buttondown）
- 读者评论（Giscus）
