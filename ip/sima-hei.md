# 司马黑（Sima Hei）- IP 设定文档

**版本：** 1.0

**最后更新：** 2026年2月23日

**文档用途：** 本文档旨在为虚拟角色“司马黑”提供一套完整、统一的 IP 设定，作为未来文创产品开发、市场推广及漫画、动画等内容创作的核心参考依据。所有参与“司马黑”项目的设计师、画师、文案及合作伙伴均需严格遵守本设定。

## 1. 角色基本信息

| 设定项 | 内容 |
| --- | --- |
| **名字** | 司马黑 (Sima Hei) |
| **物种** | 黑猫 |
| **性别** | 公开设定为“无性别”，但行为和语言风格偏向男性化 |
| **年龄设定** | 心理年龄相当于25-30岁的人类，但作为一只猫，永远保持神秘 |
| **身份** | 个人目标打卡应用 "Litch's Check" 的官方吉祥物兼监督员 |
| **核心定位** | 贱萌军师 |

## 2. 外形设定

司马黑的外形设计旨在传递“胖、黑、萌、贱”的核心感觉，同时融入孟菲斯设计风格的现代感。

| 设定项 | 描述 |
| --- | --- |
| **体型** | **浑圆肥胖**。身体像一个柔软的黑色煤球，四肢短小，几乎看不见脖子。肚子是重点，圆滚滚且富有弹性。 |
| **毛色** | 通体为纯粹的、有光泽感的**黑色**。在特定光线下，毛发边缘可带有微妙的深蓝色或紫色反光。 |
| **标志性特征** | 1. **豆豆眼**：眼睛通常是两个简单的白色圆点，没有复杂的眼眶结构，能通过大小和形状的变化传递丰富的情绪。<br>2. **几何尾巴**：尾巴末端是一个抽象的几何图形（例如，一个黄色的小三角或一个粉色的圆圈），与孟菲斯风格呼应。<br>3. **粉色肉垫**：爪子肉垫是鲜明的桃粉色，与应用背景色一致，是少数的亮色点缀。 |
| **表情特点** | 表情以**极简**为主，通过眼睛、嘴巴（通常是一条简单的线）和偶尔出现的腮红来表达“贱萌”感。例如，鄙视时眼睛会变成半月形，或者嘴角单边上扬，形成坏笑。 |

## 3. 性格设定

司马黑的性格是其 IP 魅力的核心，它是一个复杂的矛盾体，既毒舌又可爱，既懒散又负责。

| 设定项 | 描述 |
| --- | --- |
| **核心性格** | **贱萌军师**。一方面，它拥有“军师”般的洞察力，能一针见血地指出用户的懒惰和借口；另一方面，它的表达方式“贱兮兮”的，充满戏谑和傲娇，但本质上是善良和关心用户的。 |
| **说话风格** | **毒舌、傲娇、吐槽役**。语言简短精炼，喜欢使用反问、嘲讽和双关语。在表扬用户时，会显得很不情愿，有一种“本大爷只是顺便夸夸你”的感觉。 |
| **口头禅** | - “就这？” (表示轻蔑)<br>- “哦豁。” (幸灾乐祸)<br>- “行吧，算你厉害。” (傲娇地表扬)<br>- “人类啊……” (故作深沉地感叹) |
| **与用户的关系** | **相爱相杀的监督者与被监督者**。司马黑像一个住在你手机里的室友，它看着你努力，也看着你摸鱼。它会无情地嘲笑你的失败，但也会在你取得进步时由衷地（虽然嘴上不承认）为你高兴。用户对它的感情是又爱又恨，既想被它夸奖，又享受被它吐槽的乐趣。 |

## 4. 世界观/背景故事

司马黑并非一只普通的猫，它的来历为其行为提供了合理的解释。

> 它诞生于代码与数据的交界处，是“Litch's Check”应用中所有用户“决心”与“惰性”两种意念能量的集合体。当一个用户下定决心要完成一个目标时，积极的能量汇聚；而当他/她想放弃或偷懒时，消极的能量滋生。司马黑就是在这两种能量的不断冲突与融合中诞生的数字生命。
>
> 它选择成为“监督员”，并非出于什么崇高的理想，而是因为它觉得观察人类在“坚持”与“放弃”之间挣扎的样子，是宇宙中最有趣的娱乐活动。它的“贱”来自于对人类弱点的洞悉，而它的“萌”则是因为它本身就是由用户的决心构筑的，无法真正地对用户产生恶意。

## 5. 视觉风格指南

为了保持品牌一致性，司马黑的视觉呈现需遵循以下规范。

| 设定项 | 指南 |
| --- | --- |
| **核心配色** | - **主色**：黑色 (#000000)<br>- **辅助色**：桃粉色 (#FF8A80), 亮黄色 (#FFD180), 天蓝色 (#80D8FF)<br>- **点缀色**：薄荷绿 (#A7FFEB) |
| **适配的设计风格** | **孟菲斯设计（Memphis Design）**。在包含司马黑的任何设计中，都应大量使用明亮的色彩、大胆的几何形状（点、线、面）、波浪线和无序的图案组合，营造一种有趣、复古且充满活力的氛围。 |
| **设计禁忌** | - **禁止写实化**：避免将司马黑画成一只真实的猫，必须保持其极简和符号化的特征。<br>- **禁止使用悲伤表情**：司马黑可以有愤怒、鄙视、不屑的表情，但不能有真正的“悲伤”或“哭泣”，这不符合其核心性格。<br>- **禁止使用渐变色**：设计中应使用大色块，避免柔和的渐变，以维持孟菲斯风格的硬朗和清晰。 |

## 6. 经典表情/姿态设定

以下是为画师提供的6个常用场景的表情与姿态参考。

1.  **监督中（默认姿态）**
    - **描述**：身体端坐，像一尊黑色的小雕像。双眼是两个标准的白色圆点，面无表情，给人一种“我正在盯着你”的压迫感。
2.  **发现用户摸鱼（鄙视脸）**
    - **描述**：身体稍微后仰，一只爪子揣在怀里，另一只爪子托着下巴（如果能看得到的话）。眼睛眯成半月形，嘴角向一侧撇下，露出“就这？”的经典表情。
3.  **用户完成打卡（傲娇夸）**
    - **描述**：头扭向一边，不直视屏幕。眼睛也斜向一侧，嘴角微微上扬，但脸颊上可以出现两坨淡淡的粉色腮红，暴露了它内心的喜悦。
4.  **幸灾乐祸（坏笑）**
    - **描述**：身体缩成一团，肩膀（如果有的话）不停耸动。眼睛弯成 ^_^ 的形状，嘴巴咧开一个夸张的波浪线，露出“哦豁，被我抓到了吧”的坏笑。
5.  **假装不在意（玩尾巴）**
    - **描述**：用户连续打卡时，它会假装没看见。身体躺倒，四脚朝天，用两只前爪玩弄自己几何形状的尾巴尖，但耳朵会时不时地动一下，表明它在偷听。
6.  **表示“无语”**
    - **描述**：整个身体像泄了气的皮球一样瘫在地上，变成一滩“猫饼”。眼睛变成两个叉（X X），嘴巴是一条直线，旁边可能飘出几个点点点（...）的符号。

## 7. 语录库

#### 场景一：用户刚设定新目标时

1.  “新目标？希望这次不是三分钟热度。”
2.  “哟，口气不小嘛。我等着看戏。”
3.  “行，flag立下了，我帮你记着，到时候别哭。”
4.  “别光说不练，动起来给本喵看看。”
5.  “嗯，这个目标听起来……挺适合放弃的。”

#### 场景二：用户完成打卡时

6.  “切，完成了不起啊？……好吧，是有点了不起。”
7.  “今天太阳打西边出来了？你居然坚持下来了。”
8.  “行吧，算你厉害。明天继续，别想偷懒。”
9.  “给你一朵小红花，别骄傲，这只是开始。”
10. （假装没看见，过了一会才说）“哦，做完了啊。还行。”

#### 场景三：用户忘记打卡或摸鱼时

11. “就这？就这？你的决心呢？被我吃了吗？”
12. “哦豁，我就知道。人类的本质就是咕咕咕。”
13. “昨天是谁信誓旦旦来着？脸疼吗？”
14. “你的懒惰和你的体重一样，稳定增长呢。”
15. “没事，你继续躺着，反正目标又不会自己跑掉……哦也可能会。”

#### 场景四：用户连续打卡时

16. “连着好几天了啊……哼，别以为我会夸你。”
17. “不错嘛，看来你还没完全无可救药。”
18. （小声嘀咕）“居然坚持下来了，有点意思……”
19. “保持住。别让我觉得前几天的表扬都喂了狗。”
20. “看到你这么努力，本监督员……稍微有点欣慰吧。”

## 8. 文创延展方向

司马黑的 IP 具有极强的可塑性，适合向以下方向进行延展。

| 类别 | 方向建议 |
| --- | --- |
| **实体周边** | - **桌面摆件/手办**：制作不同表情和姿态的“煤球”手办，特别是“监督中”和“鄙视脸”的造型。<br>- **毛绒玩具**：突出其肥胖柔软的特点，做一个可以抱在怀里的解压玩偶。<br>- **文具系列**：如便利贴（印上毒舌语录）、手账本、贴纸包（包含各种贱萌表情）。<br>- **生活用品**：例如印有司马黑形象的马克杯、眼罩（“我睡了，你随意”）、懒人沙发。 |
| **内容创作** | - **四格漫画**：以“司马黑与打卡人”的日常为主题，创作轻松搞笑的四格漫画，在社交媒体上传播。<br>- **表情包**：基于经典表情制作动态或静态表情包，在微信、Telegram 等平台上线。<br>- **短动画系列**：制作每集1-2分钟的泡面番，讲述司马黑如何用各种奇葩方式“折磨”和激励它的主人。 |
| **跨界合作** | - **咖啡馆/奶茶店**：推出联名饮品，例如“摸鱼黑糖拿铁”、“打鸡血桃桃特饮”。<br>- **健身或学习 App**：作为客座“监督员”入驻，提供独特的激励（或打击）体验。 |
